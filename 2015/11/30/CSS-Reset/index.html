简介

CSS Reset 主要用来解决各种浏览器之间由于内置样式不一致而导致的显示问题，比如默认的 line-height, margin, font-size 等等。为了更好地达到预期的显示效果，从而引入了 CSS Reset 的解决方案，下面罗列出比较流行的 3 种解决方案。

Eric Meyer’s CSS Reset

这是 CSS Web 标准专家 Eric Meyer 的解决方案，发布于 2011.01.26。

HTML5 Doctor’s CSS Reset

这是 Richard Clark 基于 Eric Meyer 释出的早期 CSS Reset 版本做出的修改版，发布于 2010.09.17。

Yahoo YUI’s CSS Reset

Yahoo YUI 实验室发布的 CSS Reset 解决方案。

以上就是几种比较流行的 CSS Reset 解决方案，读者可以依此做出适合自己的版本，以适应各种浏览器平台的 Web 开发。
